EM alternates between estimating latent-variable posteriors and updating parameters to maximize the expected complete-data log-likelihood.
Responsibilities are posterior probabilities that each component generated each data point.
Each E-step constructs a lower bound touching the current likelihood, and each M-step improves that bound.
K-means uses hard assignments, while EM for GMMs uses soft probabilistic assignments and maximizes likelihood.
Monitor the observed-data log-likelihood, parameter changes, and possible degeneracies such as collapsing covariance matrices.
